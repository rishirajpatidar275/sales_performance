
1. Project Title / Headline

    E-Commerce Sales Performance Dashboard

      An interactive and visually rich analytics dashboard designed to monitor, analyze, and optimize sales performance across customers, products, and cities.
________________________________________
2. Short Description / Purpose
   
    The E-Commerce Sales Performance Dashboard is a Power BI report built to provide a comprehensive view of sales metrics, customer behavior, and product performance. It enables stakeholders to track revenue trends, identify top customers and products, and gain actionable insights for business growth.
________________________________________
3. Tech Stack
   
    The dashboard was built using the following tools and technologies:
  
      • Power BI Desktop – Used for building interactive visualizations and dashboard design.
      • Power Query – Data cleaning, transformation, and preparation.
      • DAX (Data Analysis Expressions) – Used for KPIs such as total revenue, average order value, and customer metrics.
      • Data Modeling – Relationships created between sales, customers, products, and location datasets.
      • File Formats – .pbix / .pbit for development and .png for dashboard preview.
________________________________________
4. Data Source
   
    Kaggle 
________________________________________
5. Features / Highlights
   
    • Business Problem

      E-commerce businesses generate large volumes of transactional data, but often struggle to:
   
        •	Track real-time sales performance
   
        •	Identify top customers and products
   
        •	Understand regional sales distribution
   
        •	Monitor repeat customer behavior
   
      Without a centralized dashboard, decision-making becomes slow and inefficient.
________________________________________
• Goal of the Dashboard

    To build an interactive tool that:
      •	Provides a holistic view of sales performance 
      •	Helps identify high-value customers and products 
      •	Tracks monthly sales trends 
      •	Enables city-level performance analysis 
      •	Supports data-driven decision-making

=> Customer Insights: Identify top customers contributing maximum revenue and target them for retention strategies.
